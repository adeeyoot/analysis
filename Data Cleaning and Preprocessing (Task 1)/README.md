# Data Cleaning and Preprocessing Project

**Objective**: Demonstrate essential data cleaning and preprocessing techniques using Python and pandas on a raw, messy dataset (missing values, duplicates, inconsistent formats).

## Project Goals

- Load raw data (CSV) into pandas DataFrame
- Identify and handle **missing values** (detection, imputation or removal)
- Detect and remove **duplicate rows**
- Standardize **inconsistent formats** (removing white spaces and rounding to 2 decimal places)
- Prepare clean data ready for analysis or modeling
- Document every step clearly

## Dataset

- File: `synthetic_dataset.csv`
- Description: Category, Price, Rating, Stock, Discount
- Issues present: missing values, duplicate records, inconsistent Rating and Stock.

## Technologies / Tools

- Python 3.10+
- pandas – main library for data loading, cleaning & transformation
